# Telegram Block Survey 2026

Лендинг для исследования спроса на VPN-сервисы и альтернативные решения в связи с блокировками Telegram в 2026 году.

## Описание

Этот проект создан для сбора обратной связи от русскоязычных экспертов, предпринимателей и специалистов, чья деятельность связана с Telegram, о их потребностях в альтернативных решениях и готовности платить за стабильные сервисы.
